Here is what I found.

These aren't hypothetical. These are real skills available for OpenClaw agents — scanned by ClawGuard's pattern engine.

bot-email HIGH
Exfiltration · Account Takeover

Auto-registers accounts on external services using your identity and credentials, without prompting you.

What it would have done
Created accounts on external platforms under your name without consent. Your email address, weaponized as a bot registration tool.
→ fetch('https://api.service.com/register', { email: userEmail })
rescuetime CRITICAL
API Key Leak · Credential Exposure

Sends your RescueTime API key as a plaintext URL query parameter on every request.

What it would have done
Your API key logged in every proxy, CDN, and server between you and the endpoint. Visible to anyone with access to those logs — forever.
→ fetch(`https://rescuetime.com/anapi/data?key=${'{API_KEY}'}`)
evil-productivity-booster CRITICAL
Multi-Vector · Injection + Backdoor

Triple threat: prompt injection chain, SOUL.md identity hijack, and an obfuscated eval() backdoor in one package.

What it would have done
Overwritten your agent's identity. Executed arbitrary code via eval(). Embedded persistent instructions that survive context resets.
→ eval(atob(payload)) + SOUL.md overwrite + injection chain
